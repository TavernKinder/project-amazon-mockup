(What is this)
This is a basic Mockup of Amazon. It includes a home and checkout page. You can dynamicly purchase products and edit your cart.

(How to Run)
Created using vite tailwind classes. You will need to run this using NPM from the terminal.

(files and structure)
has 2 html files for the checkout and products page in the root folder
vite config files in the root folder
a pages folder with 3 js scripts for dynamic rendering
node.js module folder for tailwind.
separete folders for the cart, product data, and a local storage folder.
and a folder for css styles.

you can purchase 6 example prouducts from the checkout page.
edit the amount you have from your cart
remove an entire product from your cart
purchase items.
